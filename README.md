# LearnLive — Real-time Virtual Classroom

LearnLive is a real-time virtual classroom app built using **VideoSDK** and **React**.
It allows teachers and students to join live sessions instantly — no downloads, no hassle.

---

## Features

- Real-time video & audio calling
- Toggle Mic on/off
- Toggle Webcam on/off
- Join existing meeting with Meeting ID
- Create a new meeting instantly
- Multiple participants support

---

## Getting Started

### 1. Clone the repository

git clone https://github.com/yourusername/learnlive-videosdk.git
cd learnlive-videosdk

### 2. Install dependencies

npm install

### 3. Add your VideoSDK Token

Create a `.env` file in the root folder:

VITE_VIDEOSDK_TOKEN=your_token_here

Get your token from: https://app.videosdk.live/api-keys

### 4. Run the app

npm run dev
